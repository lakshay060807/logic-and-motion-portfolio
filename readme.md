# Logic & Motion | Full-Stack Portfolio

A cinematic, typography-driven portfolio developed as part of my first-year BSc AIML coursework. This project demonstrates the integration of a minimalist frontend with a functional Node.js backend and MongoDB database.

## 🚀 Tech Stack
- **Frontend:** HTML5, CSS3 (Custom animations), JavaScript (ES6+)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (via Mongoose)
- **Visuals:** DaVinci Resolve (Cinematic motion & color grading)

## 🛠️ Key Features
- **Dynamic Hero Section:** High-fidelity background video integration.
- **Secure Contact System:** Functional backend route to capture and store user inquiries in MongoDB.
- **Responsive Design:** Modular grid layouts for project showcases and photography.
- **Security:** Use of `.env` for protected environment variables and `.gitignore` for clean version control.

## 💻 Local Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env` file and add your `MONGO_URI`.
4. Run `node server.js` to start the backend.
5. Open `index.html` in your browser.
